## 📖 About

This portfolio was designed and developed from scratch using HTML, CSS, and JavaScript. It serves as a central location to showcase my work, resume, technical skills, and contact information.

The website features a modern dark-themed design with responsive layouts, custom animations, interactive elements, and a focus on clean user experience.

---

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern dark UI with neon purple accents
- Animated hero section
- Interactive project cards with hover effects
- Skills section with technology icons
- Resume download
- Contact section with copy-to-clipboard email
- Smooth scrolling navigation
- Animated background effects
- Scroll progress indicator
- Custom TrueType fonts using `@font-face`
